# Evaluación con el cuaderno de Séneca (REA - Edición 2026)

Este repositorio contiene las fuentes y el código del **Recurso Educativo Abierto (REA): "Evaluación con el cuaderno de Séneca"**. 

Este REA ha sido diseñado como una guía metodológica y práctica orientada al profesorado de la comunidad autónoma de Andalucía, con el objetivo de facilitar la planificación, configuración y aplicación de la evaluación formativa y sumativa a través de la herramienta oficial "Cuaderno de Séneca".

* **Acceso a la versión web:** [https://juanmale14.github.io/REA-cuaderno-S-neca-2026./](https://juanmale14.github.io/REA-cuaderno-S-neca-2026./)
* **Autor:** Juanma López Esparrell (https://blogsaverroes.juntadeandalucia.es/ellocodelamochila/portada/)
* **Herramienta de autoría:** [eXeLearning](https://exelearning.net/)

## 📝 Descripción del Proyecto

Este recurso se estructura a lo largo de un itinerario formativo dividido en varias fases que acompañan al docente desde la base teórica (currículo y diseño de la evaluación) hasta el uso instrumental de la plataforma Séneca en el aula:

1. **Fase de planificación:** El currículo, el contexto de partida, los centros de interés y el plan de trabajo.
2. **Fase de configuración:** Gestión de permisos, métodos de calificación, configuración de evaluaciones, parámetros y trazabilidad.
3. **Fase de diseño:** Creación y estructuración de la propuesta didáctica.
4. **Fase de aplicación en el aula:** Diseño de instrumentos (rúbricas, cuestionarios), toma de datos diaria, visualización de pantallas clave y generación de informes de evaluación.
5. **Fase de ampliación:** Resolución de dudas a través de una sección de Preguntas Frecuentes (FAQ).

## 🛠️ Estructura del Repositorio y Tecnologías

El recurso ha sido exportado en formato de sitio web HTML desde **eXeLearning** para facilitar su alojamiento directo en GitHub Pages.

* **`/html`**: Páginas de contenido estructurado del REA.
* **`/content` y `/libs`**: Recursos multimedia, imágenes, hojas de estilo (CSS) y scripts necesarios para la visualización e interactividad del recurso.
* **`index.html`**: Punto de entrada del recurso.
* **Archivo fuente (`.elp`)**: Puedes encontrar el archivo editable de eXeLearning en la sección correspondiente de descargas dentro de la web o en este repositorio para que puedas modificarlo y adaptarlo a tu centro o aula.
